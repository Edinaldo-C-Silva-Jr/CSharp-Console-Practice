#### Português (BR)

O quarto exercício da lista. Este exercício pede para listar algumas datas importantes da área da tecnologia, utilizando diferentes modos de formatação de datas.

A aplicação mostra ao usuário os formatos de data disponíveis (usando a data de hoje como exemplo), e solicita qual formato deve ser usado. Após a escolha do formato, a aplicação informa as datas disponíveis, e solicita qual delas deve ser exibida. Por fim, é exibida a data solicitada com o formato escolhido, além de uma descrição do evento que ocorreu neste dia.

Há dois desafios opcionais no exercício: Disponibilizar novos formatos de data para serem exibidos, e utilizar a classe CultureInfo para formatar a data em culturas diferentes.

Os conceitos utilizados neste exercício são a classe DateTime, os diferentes tipos de formatação de data disponíveis em strings, e o desafio opcional utiliza a classe CultureInfo.

#### English

The fourth exercise of the list. This exercise asks to list some important dates from the technology field, utilizing diffferent methods of formatting dates.

The application shows the user the date formats available (using today's date as an example) and asks what format should be used. Then it informs the available dates, and asks which of them should be shown. Finally, the application shows the requested date with the chosen format, along with a description of the event that happened on said date.

There are two optional challenges: Offer new date formats to be shown in the application, and utilize the CultureInfo class to format the date in different cultures.

The concepts utilized in this exercise are the DateTime class, the different types of string formatting for dates, and the challenge utilizes the CultureInfo class.
